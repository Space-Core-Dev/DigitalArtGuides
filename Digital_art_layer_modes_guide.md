# 1. CHEAT SHEET — Layer Modes & Features

## Darkening Modes
- **[Multiply](ca://s?q=Explain_Multiply_layer_mode)** — darkens while preserving hue (shadows)
- **[Linear Burn](ca://s?q=Explain_Linear_Burn)** — stronger darkening (dramatic shading)
- **[Color Burn](ca://s?q=Explain_Color_Burn)** — high‑contrast darkening (stylized shadows)

## Lightening Modes
- **[Screen](ca://s?q=Explain_Screen_layer_mode)** — soft lightening (highlights)
- **[Add / Linear Dodge](ca://s?q=Explain_Add_layer_mode)** — intense glow (magic, neon)
- **[Color Dodge](ca://s?q=Explain_Color_Dodge)** — bright + saturated (fire, bloom)

## Contrast Modes
- **[Overlay](ca://s?q=Explain_Overlay_layer_mode)** — boosts contrast (color grading)
- **[Soft Light](ca://s?q=Explain_Soft_Light)** — gentle contrast (skin, soft shading)
- **[Hard Light](ca://s?q=Explain_Hard_Light)** — strong contrast (stylized lighting)

## Color Modes
- **[Hue](ca://s?q=Explain_Hue_blend_mode)** — changes hue only  
- **[Color](ca://s?q=Explain_Color_blend_mode)** — applies hue + saturation  
- **[Saturation](ca://s?q=Explain_Saturation_blend_mode)** — modifies saturation  
- **[Luminosity](ca://s?q=Explain_Luminosity_blend_mode)** — modifies brightness  

## Layer Features
- **[Alpha Lock](ca://s?q=Explain_Alpha_Lock)** — paint only on existing pixels  
- **[Clipping Mask](ca://s?q=Explain_Clipping_Mask)** — non‑destructive shading  
- **[Layer Mask](ca://s?q=Explain_layer_mask)** — non‑destructive erasing  
- **[Adjustment Layer](ca://s?q=Explain_adjustment_layers)** — color/tonal edits  
- **[Vector Layer](ca://s?q=Explain_vector_layers)** — crisp, editable strokes

# 4. BEGINNER‑FRIENDLY GUIDE

## Shadows
Use **[Multiply](ca://s?q=Explain_Multiply_layer_mode)** for:
- clothing folds  
- ambient occlusion  
- soft shadows  
**Tip:** opacity 20–40%.

## Highlights
- **[Screen](ca://s?q=Explain_Screen_layer_mode)** → soft light  
- **[Add](ca://s?q=Explain_Add_layer_mode)** → strong glow  

## Color Grading
- **[Overlay](ca://s?q=Explain_Overlay_layer_mode)** → contrast boost  
- **[Soft Light](ca://s?q=Explain_Soft_Light)** → subtle mood  

## Clean Workflow
- **[Clipping Mask](ca://s?q=Explain_Clipping_Mask)** → shading without damaging base  
- **[Alpha Lock](ca://s?q=Explain_Alpha_Lock)** → recolor shapes  
- **[Layer Mask](ca://s?q=Explain_layer_mask)** → erase non‑destructively  

## Lineart
Use **[Vector Layers](ca://s?q=Explain_vector_layers)** (Fresco, CSP) for:
- comics  
- manga  
- clean strokes
