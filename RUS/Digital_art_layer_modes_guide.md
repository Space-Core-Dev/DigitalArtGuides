# 1. ШПАРГАЛКА — Режимы слоёв и функции

## Режимы затемнения
- Multiply — затемняет, сохраняя оттенок (тени)
- Linear Burn — сильное затемнение (драматичные тени)
- Color Burn — контрастное затемнение (стилизованные тени)

## Режимы осветления
- Screen — мягкое осветление (блики)
- Add / Linear Dodge — яркое свечение (магия, неон)
- Color Dodge — яркое и насыщенное свечение (огонь, блум)

## Контрастные режимы
- Overlay — усиливает контраст (цветокоррекция)
- Soft Light — мягкий контраст (кожа, мягкие тени)
- Hard Light — сильный контраст (стилизованный свет)

## Цветовые режимы
- Hue — меняет только оттенок
- Color — изменяет оттенок и насыщенность
- Saturation — изменяет насыщенность
- Luminosity — изменяет яркость

## Функции слоёв
- Alpha Lock — рисование только по существующим пикселям
- Clipping Mask — недеструктивное затемнение/осветление
- Layer Mask — недеструктивное стирание
- Adjustment Layer — цветовые и тоновые корректировки
- Vector Layer — чёткие редактируемые линии

# 4. ГАЙД ДЛЯ НОВИЧКОВ

## Тени
Используй Multiply для:
- складок одежды
- ambient occlusion
- мягких теней  
**Совет:** непрозрачность 20–40%.

## Блики
- Screen — мягкий свет
- Add — яркое свечение

## Цветокоррекция
- Overlay — усиление контраста
- Soft Light — мягкая атмосфера

## Чистый рабочий процесс
- Clipping Mask — тени/свет без порчи основы
- Alpha Lock — перекраска формы
- Layer Mask — аккуратное стирание

## Лайнарт
Используй Vector Layers (Fresco, CSP) для:
- комиксов
- манги
- чистых линий
